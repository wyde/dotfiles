## Installation (optional)
```
    $ cd ~
    $ git clone git@github.com:wyde/dotfiles.git
    $ mv ~/dotfiles/.bashrc ~/.bashrc
    $ mv ~/dotfiles/.vimrc ~/.vimrc
```
